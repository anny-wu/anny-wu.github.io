Projects 项目经历

Disease Classification Tool (Python)

In a simulated industrial project, I first performed one-hot encoding and PCA feature reduction to prepare the dataset for model training. Then, I trained supervised (Perceptron, Logistic Regression, Decision Tree, Random Forest, K-Nearest Neighbors, Gradient Boosting, Neural Network) and unsupervised (K-means clustering) models using scikit learn, pandas, and numpy libraries. The best models achieved over 90% accuracy in predicting a patient’s diagnosis based on a list of symptoms.

(Download)

AvoidAsteroid (HTML, CSS, JavaScript)

I used jQuery to create an asteroid game where asteroids spawn randomly from different directions outside of the board. Shields and portals spawn at certain fixed time intervals and allow the player to level up. Users are given a tutorial when they start the game and are able to choose their difficulty levels that correspond to the spawn rate of asteroids.

GeoQuiz (Java, XML)

I implemented a trivia application using Android Studio that allows users to respond to a statement. It provides the user three opportunities to "cheat," and it will calculate the score when a round of questions is done. It has different user interface for the portrait and landscape orientation of the phone and is adaptable with different screen size.

Artist Search (HTML, CSS, JavaScript)

I used Vue to create an artist search website that fetches data from the iTunes API. It allows the users to search an artist and filter the result based on categories. It also enables users to play a sample of music and view all the songs grouped by collection names.

Construct (Swift)

This application is implemented using SwiftUI and Firebase and designed to facilitate the communication between construction workers and their project managers. Our team identified that they have been using iMessages for all communication tasks and were often confused because of the amount of information they received. Therefore, we target smaller, local construction businesses that could benefit from an application that creates a single source of information for their employees on their projects and improves project visibility.

Recipe Book (HTML, CSS, JavaScript)

I used Vanilla JS, jQuery, and Vue to create a recipe website that facilitates the publication of recipes of a local company. It allows users to filter the recipes based on categories and view the specific details.

Drone Delivery (C++)

I implemented MST and TSP (Nearest Insertion Heuristic) algorithims that calculate the optimal path for a hypothetical trip that a drone makes. Branch-and-bound algorithims are used to optimize the solution.
